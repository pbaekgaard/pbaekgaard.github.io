# pbaekgaard Portfolio

Personal portfolio website - currently under construction.

## Live Site

Visit [pbaekgaard.github.io](https://pbaekgaard.github.io) to see the current progress.

## Development

To view locally:
```bash
open index.html
```

Or serve with a local server:
```bash
npx serve .
```

Or use Wrangler dev server:
```bash
npm run dev
```

## Deployment

This site is deployed to Cloudflare Pages using Wrangler.

### Prerequisites

1. Install dependencies:
```bash
npm install
```

2. Authenticate with Cloudflare:
```bash
npx wrangler login
```

### Deploy

```bash
npm run deploy
```

Or directly with wrangler:
```bash
npx wrangler pages deploy .
```

## Tech Stack

- HTML5
- CSS3 (Animations, Gradients, Modern effects)
- No frameworks - pure vanilla everything
- Cloudflare Pages for hosting
